## Vim 使用指南

vim 是非常高深的，需要好好学习使用。

### 光标移动

h j k l 就不用说了。

| 按键 | 说明 |
| --- | --- |
| ^ f | 向下翻一页 |
| ^ b | 向上翻一页 |
| ^ d | 向下翻半页 |
| ^ u | 向上翻半页 |
| n space | 向后移动 n 个字符 |
| 0 | 移动到行首 |
| $ | 移动到行尾 |
| H | 移动到屏幕最上方一行 |
| M | 移动到屏幕最中间一行 |
| L | 移动到屏幕最下方一行 |
| G | 移动到文件最后一行 |
| nG or ngg | 移动到文件的第 n 行 |
| n enter | 向下移动 n 行 |

### 查找与替换

| 按键 | 说明 |
| --- | --- |
| /word | 向下查找 word 字符串 |
| ?word | 向上查找 word 字符串 |
| n | 重复上一次的查询操作 |
| N | 与 n 反向 |
| :n1,n2s/word1/word2/g | 将 n1 到 n2 行中 word1 内容替换成 word2 |
| :n1,n2s/word1/word2/gc | 将 n1 到 n2 行中 word1 内容替换成 word2，需要用户（使用 n 或 y）确认 |

### 编辑

| 按键 | 说明 |
| --- | --- |
| x | 向后删除一个字符 |
| X | 向前删除一个字符 |
| nx | 向后删除 n 个字符 |
| dd | 删除光标所在整行 |
| yy | 复制光标所在整行 |
| p | 将内容粘贴在下一行 |
| P | 将内容粘贴在上一行 |
| J | 将当前行与下一行合并 |
| u | 复原上一个操作 |
| ^ r | 重做上一个操作 |
| . | 重复前一个操作的意思 |
